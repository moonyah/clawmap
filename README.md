# 🧸 ClawMap

내 주변 인형뽑기 매장을 찾고  
실제 유저 후기와 사진으로 방문 결정을 돕는 앱입니다.

---

## 📱 주요 기능

- 📍 내 주변 인형뽑기 매장 지도 확인
- 🧭 가장 가까운 매장 추천
- 📝 매장 후기 확인 (추천 / 애매 / 비추천)
- 📸 사진 기반 리뷰 확인
- ⭐ 즐겨찾기 기능
- ➕ 매장 제보 및 수정 요청

---

## 🛠 기술 스택

- React Native (Expo)
- TypeScript
- Supabase (DB, Storage)
- Google Maps API

---

## 🚀 실행 방법

```bash
npm install
npx expo start
```
